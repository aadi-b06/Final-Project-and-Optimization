# Final-Project-and-Optimization

ApexPlanet Capstone Web App
A responsive, performance-focused web application built with HTML, CSS, and JavaScript. Developed during my internship at ApexPlanet as a final project to integrate frontend skills.

Features

Product catalog with search, category filter, sorting, and lazy-loaded images
Shopping cart with add/remove/update, localStorage persistence, and simulated checkout
Contact form with required field validation (name, email, message)
Blog section linking to external articles
Theme toggle and sample product insertion for testing

Performance Optimizations
Inlined critical CSS
Deferred and minimized JavaScript
IntersectionObserver for image lazy loading
Service worker (Blob-based) for caching and offline resilience
Preconnect hints to reduce request latency

Compatibility & Responsiveness
Tested on Chrome, Firefox, Safari, and mobile browsers
Mobile-first layout with grid-based responsive design
Polyfills for older browser support
